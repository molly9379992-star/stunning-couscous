# SKILL 02 — Сделать пачку Reels
**Файл:** `02_make_reels_pack.md`

---

## Когда использовать
- В среду (день батч-подготовки контента нейросетями)
- Когда нужно заранее заготовить Reels на 1–2 недели
- Когда падают охваты и нужно усилить видео-контент

---

## Для чего нужен
Скилл собирает 3–5 готовых сценариев Reels по двум направлениям сразу:
- Услуги в Грозном (Instagram)
- Оборудование KAVCOSMO (Instagram)

Каждый сценарий — 15–30 секунд, с крючком, телом и призывом к действию.

---

## Входные данные
- Сколько Reels нужно (рекомендую 3–5)
- Темы (услуги / аппараты), которые нужно покрыть
- Тон недели (тёплый / экспертный / провокационный)
- Есть ли акция, которую нужно встроить

---

## Что должно получиться
Файл в `06_REELS_STORIES/` с пачкой Reels. Для каждого:
- Тема и направление
- Площадка: Instagram
- Крючок (первые 3 секунды)
- Тело сценария (раскадровка по секундам)
- Текст-обложка
- Призыв к действию
- Музыка / звук (рекомендация)
- 5–8 хештегов из `hashtags.md`

---

## Пошаговый алгоритм
1. Прочитать `platform_rules.md` и `content_style.md`
2. Прочитать `services_facts.md` или `equipment_facts.md` по теме
3. Открыть шаблон `08_ШАБЛОНЫ/reels_script_template.md`
4. Для каждой темы написать крючок (вопрос / шок / интрига)
5. Расписать раскадровку по 3-секундным блокам
6. Добавить субтитры и CTA
7. Подобрать хештеги из `hashtags.md`
8. Передать результат в `quality_check_agent.md`

---

## Какие агенты использовать
- `reels_stories_agent.md` — основной
- `hook_headline_agent.md` — для крючков
- `visual_concept_agent.md` — для раскадровки

---

## Какие файлы базы прочитать
- `00_БАЗА_KAVCOSMO/platform_rules.md`
- `00_БАЗА_KAVCOSMO/content_style.md`
- `00_БАЗА_KAVCOSMO/services_facts.md` (для услуг)
- `00_БАЗА_KAVCOSMO/equipment_facts.md` (для оборудования)
- `00_БАЗА_KAVCOSMO/forbidden_claims.md`
- `00_БАЗА_KAVCOSMO/hashtags.md`

---

## Правила площадок
**Reels — только Instagram** (для обоих направлений).
Услуги: Instagram + Avito (Avito без видео).
Оборудование: Instagram + Telegram + VK + WhatsApp.
Сайт — нигде.

---

## Что нельзя делать
- Не делать Reels длиннее 30 секунд (теряется удержание)
- Не озвучивать запрещённые фразы: «навсегда», «100%», «без боли», «без противопоказаний», «моментальный результат» (для накопительных)
- Не упоминать сайт, рассрочку без банка
- Не смешивать услуги и оборудование в одном Reels
- Не использовать CAPS на обложке
- Не использовать чужую музыку без разрешения площадки

---

## Пример команды запуска
```
Запусти скилл 02_make_reels_pack.md.
Сделай 5 Reels на следующие 2 недели:
3 по услугам (лазер, RF, карбоновый пилинг) и 2 по оборудованию (Morpheus, Ultraformer).
Тон: тёплый, экспертный. Длительность каждого — до 25 секунд.
```
