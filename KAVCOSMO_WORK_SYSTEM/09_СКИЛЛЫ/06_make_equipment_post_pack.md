# SKILL 06 — Сделать пакет постов по оборудованию
**Файл:** `06_make_equipment_post_pack.md`

---

## Когда использовать
- В дни оборудования (вторник, пятница) или в среду на батч
- При запуске нового аппарата в продажу
- При прогреве к продаже конкретной модели
- Когда нужно покрыть один аппарат сразу на 4 площадках

---

## Для чего нужен
Скилл создаёт готовый комплект продающих постов по одному аппарату KAVCOSMO
сразу для всех разрешённых площадок: Instagram, Telegram, VK, WhatsApp.
Каждый текст — под формат своей площадки.

---

## Входные данные
- Аппарат (Morpheus, Volnewmer, Ultraformer MRT, Smart Ice Blue, BBL, Союз-7, MBT LumiRay, Cryo 360)
- Целевая аудитория (косметологи, клиники, инвесторы в beauty-бизнес)
- Главная выгода для клиники (окупаемость, средний чек, поток клиентов)
- Спецоффер (если есть)
- Контакты для связи (Telegram / WhatsApp / VK / Direct)

---

## Что должно получиться
Комплект из 4 текстов:
- **Instagram пост** — 800–1500 знаков, тёплый и экспертный, с CTA в Direct
- **Telegram пост** — 1500–2500 знаков, экспертный, с подзаголовками
- **VK пост** — 1000–2000 знаков, структурированный, с разделами и эмодзи-маркерами
- **WhatsApp статус** — до 700 знаков, короткий тизер с одним CTA

Плюс 5–8 хештегов из `hashtags.md` для Instagram.

---

## Пошаговый алгоритм
1. Прочитать `platform_rules.md` и `equipment_facts.md`
2. Прочитать `prices_and_offers.md` и `content_style.md`
3. Определить главную выгоду аппарата для клиники
4. Написать Instagram пост (крючок → факт → выгода → CTA)
5. Адаптировать в Telegram (длиннее, экспертнее, с подзаголовками)
6. Адаптировать в VK (с эмодзи-маркерами и блоками)
7. Сделать WhatsApp статус (тизер + один шаг к действию)
8. Подобрать хештеги для Instagram
9. Передать результат в `quality_check_agent.md`

## Правила площадок
**Этот скилл — только для оборудования.**
Разрешено: Instagram + Telegram + VK + WhatsApp.
Запрещены: **Avito** (категорически), сайт.
Avito для оборудования не используется ни в каком виде.

---

## Что нельзя делать
- Не упоминать Avito и сайт
- Не писать «рассрочка без банка» — заменять на «наш менеджер по рассрочке свяжется и подберёт условия»
- Не писать «самые низкие цены», «лучше, чем у конкурентов»
- Не обещать конкретную окупаемость без оговорки «при нагрузке X процедур»
- Не использовать CAPS LOCK
- Не давать ссылок на сторонние ресурсы
- Не смешивать в одном посте 2 разных аппарата
