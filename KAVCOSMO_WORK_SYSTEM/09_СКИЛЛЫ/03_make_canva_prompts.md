# SKILL 03 — Сделать промпты для Canva
**Файл:** `03_make_canva_prompts.md`

---

## Когда использовать
- Когда есть готовые тексты постов и нужны визуалы
- В среду (батч-подготовка) для всей недели сразу
- Перед публикацией поста, для которого ещё нет картинки

---

## Для чего нужен
Скилл создаёт точные текстовые описания визуалов для Canva (Magic Media / Text-to-Image)
по фирменному стилю KAVCOSMO: фуксия, бирюза, белый, глянец.
Работает с двумя направлениями: услуги и оборудование.

---

## Входные данные
- Тема поста (услуга или аппарат)
- Направление (услуги / оборудование)
- Площадка (Instagram пост / Сторис / Reels-обложка / Telegram / VK / WhatsApp статус)
- Тон визуала (минимализм / эмоциональный / экспертный)
- Текст, который пойдёт поверх картинки (если есть)

---

## Что должно получиться
Для каждого поста — готовый блок:
- Описание сцены (что в кадре, кто, где)
- Цветовая палитра (фирменные цвета KAVCOSMO)
- Стиль (фото / иллюстрация / 3D / коллаж)
- Композиция (где текст, где объект)
- Формат (1080×1350 для поста, 1080×1920 для Сторис, 1080×1080 для квадрата)
- Текстовый слой (что писать поверх)

---

## Пошаговый алгоритм
1. Прочитать `brand_rules.md` (цвета и стиль)
2. Прочитать `content_style.md` (форматы)
3. Открыть `08_ШАБЛОНЫ/canva_prompt_template.md`
4. Для каждой темы составить описание сцены
5. Указать палитру: фуксия #E91E63, бирюза #1ABC9C, белый, глянец
6. Указать формат под конкретную площадку
7. Прописать текстовый слой (короткий, читаемый)
8. Передать результат в `quality_check_agent.md`

## Правила площадок
**Услуги:** визуалы только под Instagram и Avito (карточка объявления).
**Оборудование:** визуалы только под Instagram, Telegram, VK, WhatsApp статус.
Сайт — нигде.

---

## Что нельзя делать
- Не использовать чужие фото без прав
- Не выходить за фирменную палитру (фуксия, бирюза, белый, глянец)
- Не ставить запрещённые фразы на визуал («навсегда», «100%», «без боли», «без противопоказаний», «рассрочка без банка»)
- Не использовать CAPS LOCK на ключевых заголовках
- Не делать визуал слишком перегруженным (макс 2 текстовых блока)
- Не использовать визуал в Avito с ссылкой на сайт
