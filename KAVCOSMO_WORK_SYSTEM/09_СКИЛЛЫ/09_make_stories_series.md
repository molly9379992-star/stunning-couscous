# SKILL 09 — Сделать серию сторис
**Файл:** `09_make_stories_series.md`

---

## Когда использовать
- Когда нужна цепочка сторис на 1 день (3–7 слайдов)
- В субботу (день лёгкого контента — только Instagram Сторис)
- При прогреве к публикации поста / Reels / акции
- Когда нужно показать «изнанку» процесса или кейс «до/после»

---

## Для чего нужен
Скилл строит связную серию сторис, где каждый слайд ведёт к следующему,
а финальный слайд — к действию:
- для услуг: запись через Direct или Avito-чат
- для оборудования: запрос в Direct, переход в Telegram, VK или WhatsApp статус
Работает для двух направлений.

---

## Входные данные
- Тема серии (услуга или аппарат)
- Направление: услуги или оборудование
- Количество слайдов (3–7)
- Цель серии (запись / прогрев / опрос / анонс)
- Площадка: Instagram Сторис (для обоих) + WhatsApp статус (только для оборудования)

---

## Что должно получиться
Серия из 3–7 слайдов. Для каждого слайда:
- Номер и тип (тизер / факт / кейс / опрос / CTA)
- Текст на экране (короткий, до 8 слов)
- Голос за кадром / подпись (если есть)
- Визуальная подсказка (что в кадре)
- Интерактив (опрос / стикер / вопрос — где уместно)
- Финальный слайд — CTA

---

## Пошаговый алгоритм
1. Прочитать `platform_rules.md` и `content_style.md`
2. Прочитать `services_facts.md` или `equipment_facts.md` по теме
3. Определить структуру серии: тизер → раскрытие → доказательство → CTA
4. Расписать каждый слайд (текст на экране + визуал + интерактив)
5. Финальный слайд — с одним конкретным шагом
6. Подобрать визуал через `canva_prompt_agent.md`
7. Передать результат в `quality_check_agent.md`

- `canva_prompt_agent.md` — для финального визуала

---

## Правила площадок
**Услуги:** только Instagram Сторис. WhatsApp / Telegram / VK — запрещены.
**Оборудование:** Instagram Сторис + WhatsApp статус (короткий тизер).
Сайт — нигде.

---

## Что нельзя делать
- Не делать серию длиннее 7 слайдов (теряется внимание)
- Не писать на слайде больше 8 слов
- Не использовать CAPS LOCK на всех слайдах
- Не ставить «100%», «навсегда», «без противопоказаний», «без боли»
- Не писать «моментальный результат» для лазера, RF, SMAS
- Не давать ссылок на сайт
- Не делать в одной серии сторис по услугам и по оборудованию одновременно
- Не размещать WhatsApp статус по услугам

---

## Пример команды запуска
```
Запусти скилл 09_make_stories_series.md.
Тема: SMAS-лифтинг — «как проходит процедура».
Направление: услуги.
Площадка: Instagram Сторис.
5 слайдов: тизер → факт → процесс → кейс «до/после» → CTA на запись в Direct.
```
