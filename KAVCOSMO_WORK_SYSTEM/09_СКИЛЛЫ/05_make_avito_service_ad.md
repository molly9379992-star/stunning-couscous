# SKILL 05 — Сделать объявление на Avito по услуге
**Файл:** `05_make_avito_service_ad.md`

---

## Когда использовать
- Когда нужно разместить или обновить объявление на Avito
- При запуске новой услуги в Грозном
- При смене цены / акции — переписать объявление
- Когда падают звонки — обновить заголовок и текст

---

## Для чего нужен
Скилл создаёт полный комплект для одного объявления на Avito по услуге:
заголовок, основной текст, краткое описание для карточки, варианты заголовков,
советы по фото. Используется только для услуг в Грозном.

---

## Входные данные
- Услуга (лазерная эпиляция / RF / SMAS / карбоновый пилинг)
- Цена за процедуру / зону / курс
- Контактный номер для записи
- Спецоффер (если есть)
- Адрес кабинета (если хочется указать)

---

## Что должно получиться
Готовое объявление в формате:
- ЗАГОЛОВОК (до 50 знаков) — основной + 3 варианта
- ОСНОВНОЙ ТЕКСТ (300–600 знаков)
- Город: Грозный + цена + телефон
- КРАТКОЕ ОПИСАНИЕ для карточки (до 100 знаков)
- Советы по фото (что снимать)

---

## Пошаговый алгоритм
1. Прочитать `platform_rules.md` и `forbidden_claims.md`
2. Прочитать `services_facts.md` по нужной услуге
3. Прочитать `prices_and_offers.md`
4. Открыть `08_ШАБЛОНЫ/` или схему из `avito_services_agent.md`
5. Написать заголовок (до 50 знаков, без CAPS)
6. Написать основной текст: что за процедура → результат → кому подходит → цена → запись
7. Сделать краткое описание (1–2 предложения)
8. Предложить 3 варианта заголовка
9. Дать советы по фото (до/после, кабинет, аппарат, чек-лист)
10. Передать результат в `quality_check_agent.md`

## Правила площадок
**Этот скилл — только для услуг в Грозном.**
Разрешено: Avito (основа) + Instagram (для адаптации текста, если нужен пост).
Запрещены: WhatsApp, Telegram, VK, сайт.
Avito для оборудования — запрещён.

---

## Что нельзя делать
- Не использовать CAPS LOCK в заголовке
- Не писать «навсегда», «100%», «без противопоказаний», «без боли»
- Не писать «моментальный результат» для лазера, RF, SMAS (только карбоновый пилинг)
- Не писать «рассрочка без банка»
- Не ставить ссылки на сайт
- Не давать ссылки на WhatsApp, Telegram, VK
- Не атаковать конкурентов («у нас лучше, чем у…»)
- Не обещать конкретных цифр без подтверждения

```
Запись: [номер].
Дай 3 варианта заголовка и основной текст с акцентом на комфорт и охлаждение.
```
