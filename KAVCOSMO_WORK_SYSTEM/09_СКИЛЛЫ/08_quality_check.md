# SKILL 08 — Проверка качества контента
**Файл:** `08_quality_check.md`

---

## Когда использовать
- **ВСЕГДА** перед публикацией любого контента — это финальный шаг для всех остальных скиллов
- Когда чужой контент уже готов и нужна верификация
- При проверке батча перед загрузкой в отложенный постинг
- При сомнении: «можно ли это публиковать?»

---

## Для чего нужен
Скилл — это финальный фильтр всей системы KAVCOSMO.
Он проверяет любой готовый материал на:
- соответствие правилам площадок
- запрещённые фразы
- тон бренда
- структуру и CTA
- цены и офферы
- противопоказания и честность

Без этой проверки контент не публикуется.

---

## Входные данные
- Готовый текст / Reels-сценарий / Avito объявление / визуальное описание
- Направление: услуги или оборудование
- Площадка, на которой планируется публикация
- Тип контента (пост / сторис / Reels / объявление / статус)

---

## Что должно получиться
Один из двух вердиктов:

**✅ ПРОШЁЛ ПРОВЕРКУ:**
- Список того, что проверено
- Подтверждение: фраз из чёрного списка нет, площадка корректна, тон в норме

**❌ НЕ ПРОШЁЛ:**
- Список проблем с цитатами из текста
- Конкретные замены по каждому пункту
- Готовый исправленный фрагмент или предложение переписать

---

## Пошаговый алгоритм
1. Прочитать `forbidden_claims.md` целиком
2. Прочитать `platform_rules.md` целиком
3. Проверить контент по чек-листу:
   - [ ] Площадка соответствует направлению (услуги ↔ Instagram/Avito, оборудование ↔ Instagram/Telegram/VK/WhatsApp)
   - [ ] Нет ссылок на сайт
   - [ ] Нет «рассрочка без банка»
   - [ ] Нет «100%», «навсегда», «без противопоказаний», «без боли»
   - [ ] Нет «моментальный результат» для лазера, RF, SMAS
   - [ ] Нет «лечит» применительно к косметологии
   - [ ] Нет CAPS LOCK в заголовках
   - [ ] Нет ложной срочности и давления
   - [ ] Цена указана корректно (если есть)
   - [ ] Есть CTA, соответствующий площадке
   - [ ] Хештеги соответствуют теме (для Instagram)
   - [ ] Тон соответствует направлению (тёплый для услуг, экспертный для оборудования)
4. Выдать вердикт + конкретные правки

## Правила площадок (свод)
| Направление | Разрешено | Запрещено |
|---|---|---|
| Услуги в Грозном | Instagram, Avito | WhatsApp, Telegram, VK, сайт |
| Оборудование KAVCOSMO | Instagram, Telegram, VK, WhatsApp | Avito, сайт |

Instagram — общий. Avito — только услуги. Сайт — нигде.

---

## Что нельзя делать
- Не пропускать ни один пункт чек-листа
- Не давать вердикт «всё хорошо» без проверки `forbidden_claims.md`
- Не разрешать публикацию с любой запрещённой фразой, даже «маленькой»
- Не разрешать Avito для оборудования и WhatsApp/Telegram/VK для услуг
- Не пропускать ссылки на сайт
- Не пропускать «рассрочка без банка»
- Не давать общие комментарии вроде «нормально» — нужны конкретные пункты
