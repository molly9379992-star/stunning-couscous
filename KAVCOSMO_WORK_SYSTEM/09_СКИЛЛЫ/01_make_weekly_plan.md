# SKILL 01 — Сделать недельный план
**Файл:** `01_make_weekly_plan.md`

---

## Когда использовать
- В воскресенье вечером или утром понедельника
- Когда нужен план контента на 7 дней по двум направлениям
- Когда заканчивается текущая неделя и нужна следующая

---

## Для чего нужен
Скилл собирает готовый недельный план KAVCOSMO с распределением задач по дням:
услуги в Грозном (Instagram + Avito) и продажа оборудования (Instagram + Telegram + VK + WhatsApp).
Максимум 3 задачи в день, чтобы не выгорать.

---

## Входные данные
- Номер недели и даты (например, «10–16 июня 2026»)
- Фокусные услуги недели (например, лазер + SMAS)
- Фокусные аппараты недели (например, Morpheus + Ultraformer)
- Акции / офферы, которые нужно прогреть
- Особые события (праздники, выходные)

---

## Что должно получиться
Файл `01_ПЛАНЫ_НЕДЕЛИ/week_XX.md` со структурой:
- Пн — услуги (Instagram + Avito)
- Вт — оборудование (Instagram + Telegram + VK + WhatsApp)
- Ср — батч-подготовка контента нейросетями
- Чт — услуги (Reels + Сторис)
- Пт — оборудование
- Сб — лёгкий бренд-контент (только Instagram)
- Вс — анализ + план следующей недели

---

## Пошаговый алгоритм
1. Прочитать `platform_rules.md` и `content_style.md`
2. Открыть шаблон `08_ШАБЛОНЫ/weekly_plan_template.md`
3. Уточнить фокусные услуги и аппараты недели
4. Расписать задачи по дням (макс 3 на день: текст + визуал + действие)
5. Проставить площадки строго по правилам направления
6. Указать сроки публикации (утро / вечер)
7. Передать результат в `quality_check_agent.md`

## Правила площадок
**Услуги в Грозном:** только Instagram + Avito. Запрещены WhatsApp, Telegram, VK, сайт.
**Оборудование KAVCOSMO:** только Instagram + Telegram + VK + WhatsApp. Запрещены Avito и сайт.
Instagram — общий для обоих направлений. Avito — только для услуг. Сайт — нигде.

---

## Что нельзя делать
- Не ставить Avito в дни оборудования
- Не ставить WhatsApp/Telegram/VK в дни услуг
- Не упоминать сайт
- Не писать «рассрочка без банка»
- Не обещать «100% результат», «навсегда», «без противопоказаний»
- Не писать «моментальный результат» для лазера, RF, SMAS (там результат накопительный)
- Не ставить больше 3 задач на день
- Не смешивать в один день услуги и оборудование в одном посте

---

## Пример команды запуска
```
Запусти скилл 01_make_weekly_plan.md.
Сделай недельный план на 10–16 июня 2026.
Фокус недели: лазерная эпиляция + SMAS-лифтинг (услуги),
Morpheus + Ultraformer MRT (оборудование).
Акция: курс лазера на 6 сеансов со скидкой.
Максимум 3 задачи в день. Сб — только Instagram Сторис.
```
