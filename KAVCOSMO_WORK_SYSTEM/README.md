# KAVCOSMO_WORK_SYSTEM
**Простая инструкция на каждый день**

---

## 1. ЧТО ЭТО ЗА СИСТЕМА

Это твоя личная папка для работы. Здесь лежит всё, что нужно, чтобы каждый день делать контент и не теряться.

Система работает с двумя направлениями:
1. **Услуги в Грозном** — лазерная эпиляция, RF-лифтинг, SMAS-лифтинг, карбоновый пилинг
2. **Продажа оборудования KAVCOSMO** — Morpheus, Volnewmer, Ultraformer MRT, Smart Ice Blue, BBL, Союз-7, MBT LumiRay, Cryo 360

Главная идея: ты не думаешь «что мне делать». Открываешь нужный файл — и делаешь.

## 4. КАК СОЗДАТЬ ПЛАН НА НЕДЕЛЮ

**Воскресенье — день планирования.** Открываешь нейросеть и запускаешь скилл:

1. Открой `09_СКИЛЛЫ/01_make_weekly_plan.md`
2. Скопируй пример команды из раздела «Пример команды запуска»
3. Подставь даты, фокусные услуги и аппараты на свою неделю
4. Отправь команду в нейросеть
5. Получи готовый план → сохрани в `01_ПЛАНЫ_НЕДЕЛИ/week_XX.md`
6. Прогоняй через `quality_check_agent.md`

## 5. КАК СОЗДАТЬ КОНТЕНТ ПО ОДНОЙ ТЕМЕ

Когда у тебя есть одна сильная тема — её можно превратить в 4–6 публикаций.

1. Открой `09_СКИЛЛЫ/04_repurpose_one_topic.md`
2. Запиши главную мысль одним предложением
3. Выбери направление (услуги или оборудование)
4. Скилл сам адаптирует тему под все разрешённые площадки направления
5. Проверь через `quality_check_agent.md`

**Пример:** одна тема «Карбоновый пилинг — голливудское сияние» = пост Instagram + 3 Сторис + объявление на Avito.

## 6. КАК СОЗДАТЬ КОНТЕНТ ДЛЯ УСЛУГ В ГРОЗНОМ

Площадки: **Instagram + Avito**.

1. Открой `09_СКИЛЛЫ/05_make_avito_service_ad.md` — для Avito-объявления
2. Или возьми агент `07_АГЕНТЫ/instagram_services_agent.md` — для поста в Instagram
3. Напиши команду: какая услуга, цена, телефон, акция
4. Нейросеть выдаст текст → проверь через `quality_check_agent.md`

**Тон:** тёплый, женский, заботливый. Обращение на «ты» или «вы». Акцент на ощущениях и результате клиента, не на технологии.

## 7. КАК СОЗДАТЬ КОНТЕНТ ДЛЯ ПРОДАЖИ ОБОРУДОВАНИЯ

Площадки: **Instagram + Telegram + VK + WhatsApp**.

1. Открой `09_СКИЛЛЫ/06_make_equipment_post_pack.md`
2. Укажи аппарат, целевую аудиторию (косметологи / клиники), главную выгоду
3. Скилл выдаст 4 текста: Instagram, Telegram, VK, WhatsApp статус
4. Проверь через `quality_check_agent.md`

**Тон:** экспертный, B2B. Цифры, факты, окупаемость, средний чек процедуры.

**Важно:** Avito для оборудования НЕ использовать никогда.

## 8. КАК ИСПОЛЬЗОВАТЬ АГЕНТОВ

Агенты — это готовые «характеры» для нейросети. Они лежат в `07_АГЕНТЫ/`.

**Как работать:**
1. Открой нужный агент (`.md` файл)
2. Скопируй ВЕСЬ текст агента
3. Вставь в начало диалога с нейросетью (ChatGPT, Claude и т.д.)
4. Дальше — пиши свой запрос. Нейросеть будет работать в роли этого агента.

**Где найти агентов:** папка `07_АГЕНТЫ/` — там 17 файлов под разные задачи.

---

## 9. КАК ИСПОЛЬЗОВАТЬ СКИЛЛЫ

Скиллы — это готовые «рецепты» для конкретных задач. Лежат в `09_СКИЛЛЫ/`.

**Как работать:**
1. Выбери задачу (например, «сделать план на неделю»)
2. Открой соответствующий скилл (`09_СКИЛЛЫ/01_make_weekly_plan.md`)
3. Прочитай разделы «Когда использовать» и «Что должно получиться»
4. Скопируй «Пример команды запуска» → подставь свои данные → отправь в нейросеть
5. Получи результат → проверь через скилл `08_quality_check.md`

**Чем скилл отличается от агента:**
- Агент — это «кто пишет» (роль, тон, стиль)
- Скилл — это «что делаем» (пошаговый рецепт)
- В одном скилле обычно используется 1–3 агента

## 12. КАК ПРОВЕРЯТЬ ОШИБКИ ЧЕРЕЗ quality_check_agent.md

**Это финальный шаг для ЛЮБОГО контента. Без проверки — не публикуй.**

1. Открой `07_АГЕНТЫ/quality_check_agent.md`
2. Скопируй его текст в нейросеть
3. Отправь готовый текст с пометкой направления и площадки
4. Получишь один из двух вердиктов:
   - **✅ Прошёл проверку** — можно публиковать
   - **❌ Не прошёл** — список ошибок и готовых правок

**Что проверяет:**
- Соответствие площадки направлению
- Запрещённые фразы («навсегда», «100%», «без боли» и др.)
- Тон бренда
- Цены и офферы
- Наличие CTA
- Хештеги под тему

Альтернатива — скилл `09_СКИЛЛЫ/08_quality_check.md` (та же проверка, но в виде рецепта).

## 13. ЧТО ЗАПРЕЩЕНО ПИСАТЬ

❌ «Рассрочка без банка» → пиши: «наш менеджер по рассрочке свяжется и подберёт условия»
❌ «100% результат» → пиши: «видимый результат у большинства клиентов»
❌ «Без противопоказаний» → пиши: «минимальный список противопоказаний, уточним на консультации»
❌ «Навсегда» → пиши: «долгосрочный результат», «на месяцы и годы»
❌ «Моментальный результат» (для лазера, RF, SMAS — там накопительный эффект) → пиши: «результат нарастает в течение курса»
❌ «Без боли» → пиши: «минимальный дискомфорт», «есть система охлаждения»
❌ «Лечит» (применительно к косметологии) → пиши: «улучшает», «корректирует»
❌ Ссылка на сайт → не указывай вообще
❌ WhatsApp / Telegram / VK в задачах по услугам → не указывай
❌ Avito в задачах по оборудованию → не указывай

Полный список — в `00_БАЗА_KAVCOSMO/forbidden_claims.md`.
