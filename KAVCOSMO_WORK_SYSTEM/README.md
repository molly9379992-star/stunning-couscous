# KAVCOSMO_WORK_SYSTEM
**Система планирования контента и продвижения**

---

## ЧТО ЭТО И ЗАЧЕМ

Это твоя личная система для работы с двумя направлениями:
1. **Услуги в Грозном** — лазерная эпиляция, RF-лифтинг, SMAS-лифтинг, карбоновый пилинг
2. **Продажа оборудования KAVCOSMO** — Morpheus, Volnewmer, Ultraformer MRT и другие аппараты

Система устроена так, чтобы ты не думала «что делать сегодня» — просто открыла файл и сделала.

## ЧТО ОТКРЫВАТЬ КАЖДЫЙ ДЕНЬ

### Шаг 1 — Открой папку `01_ПЛАНЫ_НЕДЕЛИ/`
Найди файл текущей недели (например, `week_01_start.md`).
Посмотри, какой сегодня день и что запланировано.

### Шаг 2 — Прочитай свои 3 задачи
Каждый день — максимум 3 задачи:
- Одна задача на текст
- Одна задача на визуал (Canva)
- Одно действие (публикация или общение с клиентами)

### Шаг 3 — Возьми нужный шаблон
Шаблоны лежат в папке `08_ШАБЛОНЫ/`.
Выбери нужный под тип контента — и заполни под свою тему.

### Шаг 4 — Сохрани готовое
Готовые тексты — в папку `04_ГОТОВЫЕ_ТЕКСТЫ/`.
Готовые промпты для Canva — в `05_ПРОМПТЫ_ДЛЯ_НЕЙРОСЕТЕЙ/`.

## КАК СОЗДАТЬ НОВЫЙ НЕДЕЛЬНЫЙ ПЛАН

1. Открой воскресенье из плана текущей недели
2. Создай новый файл в папке `01_ПЛАНЫ_НЕДЕЛИ/`
   - Назови его: `week_02.md`, `week_03.md` и т.д.
3. Скопируй шаблон из `08_ШАБЛОНЫ/weekly_plan_template.md`
4. Заполни: выбери тему на каждый день, не повторяй прошлую неделю
5. Чередуй: в понедельник и четверг — разные услуги, во вторник и пятницу — разные аппараты

**Правило выбора темы:**
- Услуги: лазерная эпиляция → RF-лифтинг → SMAS-лифтинг → карбоновый пилинг → повтор
- Оборудование: выбирай тот аппарат, о котором давно не писала

## КАК ПРЕВРАТИТЬ ОДНУ ТЕМУ В НЕСКОЛЬКО ФОРМАТОВ

Это называется переупаковка контента. Работает по-разному в зависимости от направления.

**Услуги — 1 тема = 4–5 публикаций за 3 дня:**
- День 1: пост в Instagram
- День 1: объявление на Avito
- День 2: Reels в Instagram
- День 3: 3 Сторис в Instagram

**Оборудование — 1 тема = 5–6 публикаций за 3–4 дня:**
- День 1: пост в Instagram → статус в WhatsApp
- День 2: пост в Telegram → пост в VK
- День 3: Reels в Instagram
- День 3: 3 Сторис в Instagram

Шаблон для этого: `08_ШАБЛОНЫ/content_repurpose_template.md`
Агент для этого: `07_АГЕНТЫ/repurpose_content_agent.md`

## КАК РАБОТАТЬ С АГЕНТАМИ

Агенты — это готовые инструкции для нейросетей (ChatGPT, Claude и др.).
Скопируй нужный агент, вставь в нейросеть и добавь свой запрос.

| Агент | Когда использовать |
|-------|--------------------|
| `weekly_planner_agent.md` | В воскресенье — планировать неделю |
| `instagram_services_agent.md` | Пишешь контент об услуге (Instagram + Avito) |
| `equipment_sales_agent.md` | Пишешь контент об аппарате (Instagram + Telegram + VK + WhatsApp) |
| `canva_prompt_agent.md` | Нужен визуал — описание для Canva |
| `reels_stories_agent.md` | Делаешь Reels или сторис |
| `client_reply_agent.md` | Нужно ответить клиенту |
| `repurpose_content_agent.md` | Один текст → несколько форматов |

## КАК НЕ ПЕРЕГРУЖАТЬСЯ

**Правило трёх:**
Максимум 3 задачи в день. Лучше сделать 3 хорошо, чем 7 плохо.

**Правило среды:**
Среда — день заготовок. Сделай контент на несколько дней вперёд, пока есть время и настроение.

**Правило одной темы:**
Один день — одна тема. Не смешивай услуги и оборудование в один день.

**Правило переупаковки:**
Не придумывай новое каждый день. Один хороший текст даёт тебе публикации на 3–4 дня.

**Правило паузы:**
Если сегодня не получается — пропусти день. Одна пропущенная публикация не разрушит бизнес. Выгорание разрушит.
