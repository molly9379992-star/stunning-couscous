## ФОРМАТЫ ОФФЕРОВ

### Для услуг (Instagram + Avito)

**Пакетное предложение:**
```
Курс из [N] процедур [название] — [ЦЕНА] ₽
(вместо [ЦЕНА ПОШТУЧНО] ₽ за каждую)
Экономия — [СУММА] ₽
```

**Сезонный оффер:**
```
[Месяц / сезон] — самое время для [услуги].
Запишись в [период] — и [выгода].
```

**Первый визит:**
```
Первая консультация — бесплатно.
Приходи, познакомимся, подберём процедуру.
```

### Для оборудования (Instagram / Telegram / VK / WhatsApp)

**Демонстрация:**
```
Хотите увидеть аппарат в работе перед покупкой?
Организуем онлайн-показ — в удобное для вас время.
Пишите в [площадка].
```

**Окупаемость:**
```
[Аппарат] окупается при [N] процедурах в месяц.
Средний чек — от [ЦЕНА] ₽.
Расскажем подробнее — пишите.
```

## ПРАВИЛА НАПИСАНИЯ ЦЕН

✅ Писать: «от [ЦЕНА] ₽» — если цена зависит от зоны или объёма
✅ Писать: «Стоимость уточните при записи» — если не хочешь указывать точно
✅ Писать: «наш менеджер по рассрочке свяжется с вами и подберёт условия» — если речь о рассрочке

❌ Не писать: «рассрочка без банка»
❌ Не обещать «самые низкие цены в городе» без подтверждения
❌ Не указывать цену без уточнения зоны / объёма — вводит в заблуждение
❌ Не упоминать сайт как источник актуальных цен
