## СТИЛЬ ПО НАПРАВЛЕНИЯМ

### Услуги в Грозном (Instagram + Avito)

| Параметр | Как писать |
|----------|-----------|
| Тон | Тёплый, женский, заботливый |
| Обращение | «ты» или «вы» — выбери одно на весь пост |
| Длина поста | 300–600 знаков (Instagram), 300–500 знаков (Avito) |
| Акцент | Результат и ощущения, не технология |
| Локальность | Уместно упоминать Грозный |
| Призыв | «Запишись», «Напиши в Direct», «Оставь заявку» |

### Продажа оборудования (Instagram / Telegram / VK / WhatsApp)

| Параметр | Как писать |
|----------|-----------|
| Тон | Деловой, профессиональный, уверенный |
| Обращение | «вы» (профессиональная аудитория) |
| Длина поста | Instagram: 300–500 зн. / Telegram: 500–800 зн. / VK: 400–600 зн. / WhatsApp: до 150 зн. |
| Акцент | Выгода для бизнеса: окупаемость, клиенты, доход |
| Цифры | Приветствуются — средний чек, окупаемость, количество процедур |
| Призыв | «Пишите в Direct / WhatsApp / Telegram» |

## СТРУКТУРА ПОСТА

### Instagram-пост (универсальная структура)

```
[КРЮЧОК — первая фраза, останавливает скролл]

[СУТЬ — 2–3 абзаца, по делу]

[ПРИЗЫВ — одно чёткое действие]

[ХЕШТЕГИ — 5–8 штук, в конце]
```

### Telegram-пост

```
**[ЗАГОЛОВОК ЖИРНЫМ]**

[Текст с деталями и цифрами]

[Призыв + контакт]
```

### Avito-объявление

```
[ЗАГОЛОВОК — до 50 знаков, конкретно]

[Описание: что, как, результат, для кого]

📍 Город: Грозный
💰 Цена: [цена]
📞 Запись: [номер]
```

## КРЮЧКИ — КАК ПИСАТЬ ПЕРВУЮ ФРАЗУ

**Работающие приёмы:**

| Приём | Пример |
|-------|--------|
| Вопрос | «Сколько раз ты уже отложила запись на лазерную эпиляцию?» |
| Провокация | «Почему RF-лифтинг не работает — если делать его неправильно» |
| Факт / число | «6–8 сеансов — и о бритве можно забыть на год» |
| Боль клиента | «Устала от волос, которые отрастают через три дня?» |
| Результат | «После карбонового пилинга — кожа как после отпуска» |
| Для оборудования | «Один аппарат окупается при 6 процедурах в месяц» |

**Что не работает:**
- «Добрый день, дорогие подписчики!» — скука
- «Рады представить вам...» — корпоративно и холодно
- «ЗАПИШИСЬ ПРЯМО СЕЙЧАС!!!» — давление и крик

## ПРИЗЫВЫ К ДЕЙСТВИЮ

### Для услуг:
- «Пиши в Direct — запишу тебя»
- «Оставь заявку на Avito — перезвоню»
- «Напиши в Direct, расскажу подробнее»
- «Запишись на консультацию — первая бесплатно»

### Для оборудования:
- «Пиши в Direct — расскажем о стоимости и условиях»
- «Напишите в WhatsApp / Telegram — организуем онлайн-показ»
- «Свяжитесь с нами — подберём аппарат под задачи клиники»

**Правило одного призыва:**
В одном посте — один призыв. Не пиши: «Запишись / позвони / напиши / зайди на сайт».
Выбери одно действие и скажи его чётко.

## ХЕШТЕГИ — ОБЩИЕ ПРАВИЛА

- 5–8 хештегов на Instagram-пост
- Размещать в конце поста
- Хештеги должны быть релевантными теме поста
- Не использовать случайные популярные хештеги не по теме
- Подробный список — в файле `hashtags.md`
