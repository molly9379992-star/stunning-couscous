# ШАБЛОН ПРОМПТОВ ДЛЯ CANVA

---

## КАК ИСПОЛЬЗОВАТЬ
1. Открой Canva
2. Выбери нужный размер (см. таблицу ниже)
3. Следуй описанию: фон → фото → текст → детали
4. Сохрани в папку `05_ПРОМПТЫ_ДЛЯ_НЕЙРОСЕТЕЙ/` для повторного использования

---

## РАЗМЕРЫ И ГДЕ ИСПОЛЬЗУЕТСЯ

| Размер | Где |
|--------|-----|
| 1080×1080 | Instagram-пост, VK-пост |
| 1080×1920 | Instagram Stories, VK История, WhatsApp статус |
| 1280×720 | Обложка Reels, превью видео |
| 1200×628 | Telegram (горизонтальная картинка) |

---

## СТИЛЬ БРЕНДА KAVCOSMO

**Цвета:**
- Основной фон: белый (#FFFFFF) или нежно-розовый (#FFF0F0)
- Акцент: золотой (#C9A84C) или розово-золотой (#D4A9A9)
- Текст: тёмно-коричневый (#3D2B1F) или чёрный (#1A1A1A)
- Дополнительно: бежевый (#F5EDE3), серо-лиловый (#D6D0E0)

**Шрифты:**
- Заголовок: крупный, современный без засечек (Montserrat, Nunito, Raleway)
- Текст: читаемый, средний (Lato, Open Sans)

**Стиль:**
- Минималистичный, premium beauty
- Максимум 2–3 элемента на одном экране
- Много воздуха — не перегружать

## ШАБЛОН ОПИСАНИЯ ДЛЯ CANVA

```
🖼 ВИЗУАЛ ДЛЯ: [площадка + тип контента]
📐 РАЗМЕР: [px × px]

ФОН: [цвет / градиент / фото]
ГЛАВНОЕ ФОТО: [что изображено / откуда взять]
ОСНОВНОЙ ТЕКСТ (крупно): «[точный текст]»
ВТОРОСТЕПЕННЫЙ ТЕКСТ (мелко): «[дополнительный текст]»
ЦВЕТА: основной [цвет], акцент [цвет], текст [цвет]
ДОПОЛНИТЕЛЬНО: [рамка / иконка / линия / лого]
```

## ГОТОВЫЕ ПРИМЕРЫ

### Пример 1 — Пост об услуге (Instagram 1080×1080)
```
ФОН: белый с нежно-розовым градиентом снизу
ФОТО: женское лицо крупным планом, чистая кожа, мягкий свет — слева
КРУПНЫЙ ТЕКСТ: «Карбоновый пилинг»
МЕЛКИЙ ТЕКСТ: «Чистая кожа за 1 процедуру. Грозный»
ЦВЕТА: фон белый, текст тёмно-коричневый, акцент золотой
ДОПОЛНИТЕЛЬНО: тонкая золотая линия под заголовком, лого внизу по центру
```

---

### Пример 2 — Сторис об оборудовании (1080×1920)
```
ФОН: тёмный — тёмно-коричневый или чёрный
ФОТО: аппарат Morpheus, студийная съёмка — по центру
КРУПНЫЙ ТЕКСТ: «Morpheus»
МЕЛКИЙ ТЕКСТ: «Микроигольчатый RF-лифтинг для вашей клиники»
ЦВЕТА: фон тёмный, текст белый, акцент золотой
ДОПОЛНИТЕЛЬНО: золотая рамка по краям, кнопка «Узнать цену» снизу
```

---

### Пример 3 — WhatsApp статус (1080×1920)
```
ФОН: нежно-розовый однотонный
ФОТО: нет (только текст)
КРУПНЫЙ ТЕКСТ: «Есть свободные окна сегодня»
МЕЛКИЙ ТЕКСТ: «Лазерная эпиляция. Запишись сейчас.»
ЦВЕТА: фон розовый, текст тёмно-коричневый
ДОПОЛНИТЕЛЬНО: эмодзи 💚 рядом с текстом
```
